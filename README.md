# Turing Machine Simulator

This project is a Turing machine simulator implemented in JavaScript. It allows users to input strings and visualize the operation of a Turing machine based on defined transitions.

## Project Structure

```
turing-machine-simulator
├── src
│   ├── script.js        # Implementation of the Turing machine simulator
│   └── styles.css       # Styles for the user interface
├── public
│   └── index.html       # Main HTML document for the application
├── package.json          # npm configuration file
├── .gitignore            # Files and directories to be ignored by Git
└── README.md             # Documentation for the project
```

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/turing-machine-simulator.git
   ```
2. Navigate to the project directory:
   ```
   cd turing-machine-simulator
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

1. Open `public/index.html` in a web browser.
2. Enter two strings separated by a `|` in the input field.
3. Click the "Run" button to simulate the Turing machine.
4. Observe the steps and transitions of the machine as it processes the input.
